# 表

## 创建表

### 语法

```sql
CREATE TABLE table_name
(
    field1 datatype,
    field2 datatype,
    field3 datatype
) character set 字符集 collate 校对规则 engine 引擎;
```

**说明**

-   `field` : 指定列名
-   `datatype` : 指定列的数据类型
-   `character set` : 字符集。默认使用数据库的字符集
-   `collate` : 校对规则。默认使用数据库的校对规则
-   `engine` : 引擎
